# Основы Python

## Переменные и типы данных

В Python не нужно объявлять тип переменной - он определяется автоматически:

```python
# Числа
x = 10          # int (целое число)
y = 3.14        # float (число с плавающей точкой)

# Строки
name = "Python"
text = 'Hello'

# Логический тип
is_valid = True
is_empty = False

# None (отсутствие значения)
result = None
```

## Основные операторы

### Арифметические операторы
- `+` сложение
- `-` вычитание
- `*` умножение
- `/` деление
- `//` целочисленное деление
- `%` остаток от деления
- `**` возведение в степень

### Операторы сравнения
- `==` равно
- `!=` не равно
- `>` больше
- `<` меньше
- `>=` больше или равно
- `<=` меньше или равно

### Логические операторы
- `and` логическое И
- `or` логическое ИЛИ
- `not` логическое НЕ

## Ввод и вывод

```python
# Вывод
print("Hello, World!")

# Ввод (всегда возвращает строку)
name = input("Введите имя: ")

# Преобразование типов
age = int(input("Введите возраст: "))
```
